# Mobile Setup

目标：手机只负责提醒和快速回传，Codex 负责保存学习日志、复习队列和下一次安排。

## 推荐手机流程

1. 在 ChatGPT 手机 App、手机日历或提醒事项中创建 4 个每日提醒。
2. 每次提醒响起后，打开这个 Codex 英语家教线程。
3. 我会给出当下任务，你只完成当前时段内容。
4. 学完后用极简格式回传。
5. Codex 自动把打卡写入 `state/checkins.jsonl`，并更新 `state/review-items.json`。

## 提醒时间

| 时间 | 时段 | 时长 | 是否开口 |
| --- | --- | ---: | --- |
| 08:30 | 早地铁 | 25 分钟 | 否 |
| 12:30 | 午间 | 5-10 分钟 | 否 |
| 18:30 | 晚地铁 | 25 分钟 | 否 |
| 21:30 | 夜练 | 40 分钟 | 是 |

## 导出提醒文案

运行：

```bash
python3 english-coach/tools/coach.py reminders
```

然后把输出里的 4 段“复制到 ChatGPT”文本分别放进手机提醒。

## 极简回传格式

```text
时间段：
完成：
不熟：
不会：
```

示例：

```text
时间段：夜练
完成：A+ 第 2 课，30 秒项目进展说明
不熟：current-status
不会：explain schedule risk
```

很忙时可以只发一句：

```text
晚练完成 A+ 第 1 课，30 秒项目进展说明，blocker 不熟，schedule risk 不会
```

这类一句话会保守识别明确的完成项和薄弱项；没有 `不熟`、`不稳`、`不会`、`卡住` 标记的内容不会进入薄弱复习队列。

## ChatGPT 手机任务说明

如果使用 ChatGPT 手机 App 的任务/提醒功能，建议只让它做“叫醒和提示打开 Codex 线程”，不要把长期复习账本放在多个地方。这样学习数据不会散在 ChatGPT、日历和 Codex 之间。

OpenAI 帮助中心说明，ChatGPT 的任务功能可用于一次性或重复任务，并可通过 ChatGPT 的任务设置或通知设置管理提醒。可用性和数量限制可能随账号与产品更新变化；本地权威学习记录仍以 `english-coach` 文件夹为准。
